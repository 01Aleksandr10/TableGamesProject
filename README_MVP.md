# TFP — Поиск игроков в настольные игры

**TFP** (Tabletop Friends Project) — веб-приложение для поиска игроков и организации встреч по настольным играм.  
Позволяет быстро создать встречу, найти подходящую компанию, увидеть реальное количество участников и общаться в чате встречи.

Проект разработан в рамках учебного задания и полностью соответствует основным требованиям технического задания.

## Основные возможности

- Регистрация и авторизация (JWT)
- Каталог настольных игр
- Создание встреч (оффлайн / онлайн, с указанием времени и места)
- Лента встреч с фильтрами и цветовой индикацией статуса
- Присоединение / отмена участия (счётчик участников обновляется)
- Чат внутри каждой встречи
- Личный кабинет и раздел «Мои игры»
- Автоматическое заполнение тестовыми данными при первом запуске

## Технологии, используемые в проекте

**Backend**
- FastAPI — основной фреймворк
- SQLAlchemy — ORM
- SQLite — база данных (упрощено для удобства запуска)
- JWT + bcrypt — авторизация и безопасность паролей
- Pydantic — валидация данных
- Docker Compose — контейнеризация

**Frontend**
- Jinja2 + HTMX — динамика без перезагрузки страницы
- Tailwind CSS — адаптивный дизайн (Mobile First)
- Имитация мобильного приложения с Bottom Navigation Bar

**Архитектура**
- Многослойная (Layered Architecture)
- routers/ — API-эндпоинты
- schemas/ — Pydantic-модели
- db/ — модели SQLAlchemy, сессии, seed-данные
- core/ — конфигурация и безопасность
